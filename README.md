## Description

A WebExtension to mark a location on a web page and scroll to it.

## Install

<https://addons.mozilla.org/en-US/firefox/addon/mark-location/>

## Usage

- `Ctrl + , + 0-9` to mark a location.
- `Ctrl + . + 0-9` to go to a saved location.
